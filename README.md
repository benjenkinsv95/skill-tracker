## Getting Started

To get a local copy up and running follow these simple steps.

### Installation

1. Clone the repo

```sh
git clone https:://github.com/benjenkinsv95/skill-tracker.git
```

2. Install NPM packages

```sh
npm install
```

3. Start Website

```sh
npm start
```
